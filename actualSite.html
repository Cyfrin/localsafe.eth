To use a 100% local UI to interact with your safe wallet either:

1. Go to the hosted version:
https://localsafe-eth.vercel.app/
2. Run it locally yourself:
git clone https://github.com/Cyfrin/localsafe.eth cd localsafe.eth pnpm install pnpm run dev

Benefits:

  • You will not announce your transactions to the Safe team
  • You'll be able to interact with your safes even if the Safe UI goes down
  • You won't have to deal with the Safe project becoming compromised
